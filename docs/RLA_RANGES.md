# RLA Performance Level Ranges

## 8th Grade
| Level | Score Range |
|-------|------------|
| Masters | ≥70 |
| Meets | 52-69 |
| Approaches High | 43-51 |
| Approaches Low | 34-42 |
| Did Not Meet High | 27-33 |
| Did Not Meet Low | <27 |

## 7th Grade
| Level | Score Range |
|-------|------------|
| Masters | ≥75 |
| Meets | 59-74 |
| Approaches High | 50-58 |
| Approaches Low | 43-49 |
| Did Not Meet High | 32-42 |
| Did Not Meet Low | <32 |

## Usage Notes
1. These ranges are specific to RLA (Reading Language Arts)
2. Different from Math department ranges
3. Will be used to calculate performance levels from raw scores
4. Group numbers will be assigned based on combined_scores.md mapping
